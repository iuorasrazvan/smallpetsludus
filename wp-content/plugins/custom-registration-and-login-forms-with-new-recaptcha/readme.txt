Just another Custom Registration and Login Forms plugin with New Recaptcha. Simple but flexible.

== Description ==

This is Simple  Registration and  Login Forms system with Google Recaptcha. This simple plugin will show custom registration and custom login on any post or pages by just short code with recaptcha.
To generate public and private key click on this link https://www.google.com/recaptcha/admin


== Installation ==

1. Upload the entire `custom-registration-and-login-forms-with-new-recaptcha` folder to the `/wp-content/plugins/` directory.

1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Register Plugin' section at left side in your WordPress admin panel.
